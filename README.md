# GigVerse - Freelancing Platform

## 📌 Project Overview

GigVerse is a full-stack MERN freelancing platform developed to simplify collaboration between clients and freelancers. The platform enables clients to post projects, manage job listings, and review freelancer bids, while freelancers can explore opportunities, submit proposals, and track project progress.

The application focuses on providing a secure, scalable, and user-friendly freelancing environment with real-time interaction and intelligent project management features.

---

## 🚀 Features

- 🔐 Secure User Authentication & Authorization
- 👨‍💼 Role-Based Access (Client & Freelancer)
- 📢 Project Posting & Management
- 💰 Bid Submission & Tracking
- 💬 Real-Time Communication
- 📋 Freelancer Dashboard
- 📱 Responsive User Interface
- 🔎 Search & Recommendation Features
- ⚡ REST API Integration
- 🛡️ JWT-Based Authentication
- 📂 Dynamic Project & Profile Management

## 🏗️ System Architecture

The project follows a three-tier MERN architecture:

- **Frontend Layer** → React.js based responsive UI
- **Backend Layer** → Node.js & Express.js REST APIs
- **Database Layer** → MongoDB for storing users, projects, bids, and messages

The architecture ensures scalable communication between users and supports efficient project and bid management.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/mdreehal18/GigVerse-Freelancers.com.git
```

---

### 2️⃣ Navigate to Project Folder

```bash
cd GigVerse-Freelancers.com
```

---

## 🔹 Frontend Setup

### Install Dependencies

```bash
cd frontend
npm install
```

### Run Frontend

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## 🔹 Backend Setup

### Install Dependencies

```bash
cd backend
npm install
```

### Start Backend Server

```bash
npm start
```

Backend runs on:

```bash
http://localhost:5000
```
